Usage Guidelines

Hierarchy

Use only one primary button per section. Secondary and outline buttons can be used multiple times.

Accessibility

Ensure buttons have sufficient color contrast and are keyboard accessible with focus states.

Labels

Use action-oriented labels like "Download," "Subscribe," or "Learn More" instead of generic "Click Here."
